# 简介

此项目属于仿开眼项目[一款很棒的APP](http://www.kaiyanapp.com/)，开眼属于内容类应用，几乎全部是优质视频，交互设计很棒，是一个值得模仿的项目。

项目使用Kotlin编写，视频播放则是自己封装的IjkPlayer，数据来自开眼（解析需要技巧），下面的文章列表则是整个项目编写的过程。
